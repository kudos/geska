v0.5 RC1 Readme

Geska is designed to be a simple yet powerful Blogging tool licensed under the GPL.
It is very easy to set up and even easier to use with bare minimal knowledge of php and mysql needed.
One of it's major attributes is the ease of integration into existing websites, due to it's simple templating system.

Features

  • Fully functional PHP/MySQL Blog Engine.
  • Templating System allowing a choice of multiple themes(2 Wordpress ports), plus your own.
  • FCK Editor for posting blog entries.
  • Publishing system for drafting blog entries.
  • Blogger 2 API , allowing blogging from the desktop with all Blogger 2 compatible clients.
  • Multi-User capabilities.

Install

  • Extract the files.
  • Open db_config.php and make the necessary changes.
  • Place the files on your webserver .
  • chmod 777 the 'UserFiles' directory to enable uploading in FCK.
  • Open the directory in your browser.
  • Fill in the fields and click 'Install'.
  • If all goes accordingly, that's it!
